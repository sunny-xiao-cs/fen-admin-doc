# 快速开始

请确保您的操作系统上安装了

  • Vue-cli3
  • Node.js 10+
  • MYSQL 5.7+
  • Redis 6.0+

# 所需环境

这里列出项目所需的环境与相关安装教程，方便刚入门的同学

1、Node v10+
安装教程：https://www.runoob.com/nodejs/nodejs-install-setup.html
2、Redis 3.0+
安装教程：https://www.runoob.com/redis/redis-install.html
3、MYSQL 5.5.0+
安装教程：https://www.runoob.com/mysql/mysql-install.html

TIP

前端安装完 node 后，最好设置下淘宝的镜像源，不建议使用 cnpm（可能会出现奇怪的问题）

npm config set registry https://registry.npm.taobao.org
配置后可通过下面方式来验证是否成功
npm config get registry

在 ~/.npmrc 加入下面内容，可以避免安装 node-sass 失败
sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

.npmrc 文件位于
win：C:\Users\[你的账户名称]\.npmrc
linux：直接使用 vi ~/.npmrc

# 运行项目

TIP

因为本项目是前后端分离的，所以需要前后端都启动好，才能进行访问

# 数据库

启动项目前一定要在创建好 fen-web 数据库，以下是执行数据库命令：

# 登录数据库
$ mysql -uroot -p密码

# 创建 fen-web 数据库
$ CREATE DATABASE IF NOT EXISTS fen-web DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 后端运行

首先克隆项目到本地，Mac 使用终端、Windows 使用 cmd，定位到工作的目录

git clone https://github.com/sunny-xiao-cs/fen-server.git

打开VSCode，导入该目录

# 启动 Egg 后端项目
1. 在根目录下进入fen-server项目：cd fen-server
2. 安装依赖包，执行: npm install 命令，
3. 启动服务: npm run dev; 即可以启动。

# 前端运行

首先克隆项目到本地，Mac 使用终端、Windows 使用 cmd，定位到工作的目录

git clone https://github.com/sunny-xiao-cs/fen-admin.git

打开 VSCode，导入我们的项目

# 启动后台管理系统
1. 在根目录下进入fen-admin项目：cd fen-admin，
2. 安装包，执行: npm install 命令，
3. 启动服务: npm run dev; 即可以访问。

启动完后打开 localhost:8013 即可

# 前端运行常见问题

1、依赖安装失败

可通过配置淘宝的镜像源解决

npm config set registry https://registry.npm.taobao.org
配置后可通过下面方式来验证是否成功
npm config get registry

在 ~/.npmrc 加入下面内容，可以避免安装 node-sass 失败
sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

.npmrc 文件位于
win：C:\Users\[你的账户名称]\.npmrc
linux：直接使用 vi ~/.npmrc

2、linux 系统在安装依赖的时候会出现 node-sass 无法安装的问题

解决方案：

1. 单独安装：npm install --unsafe-perm node-sass
2. 直接使用：npm install --unsafe-perm
